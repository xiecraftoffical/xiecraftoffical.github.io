Part1 指令

<>代表必填 []代表选填

  1. /cd 打开主菜单，可能会显示无效指令，但是是可以用的
  2. /menu 同 /cd
  3. /warp <name> 传送到某个设置好的传送点，已经设置好的传送点有
    点击展开
    • baota 一个宝塔建筑
    • guoyuan 一个粘液科技的果园
    • hequ 腐竹闲的没逝做的车站
    • market 全球商店
    • paoku 前往跑酷世界
    • swzfarm 半成品守卫者农场
    • xiecraft 生存世界传送点
  4. /gsit 坐下
  5. /glay 躺下
  6. /gcrawl 趴下
  7. /gspin 螺~旋~冲~刺~(bushi)
  8. /spawn 回到主城
  9. /co i(nspect) 启用/关闭CoreProtect检查器
  10. /tpa <player> 请求传送到某玩家处
  11. /tpahere <player> 请求某玩家传送到你这里
  12. /tpaccept 接受传送请求
  13. /tpdeny 拒绝传送请求
  14. /tpr 随机传送
  15. /seen <player> 查看某玩家的登出时间，封禁，禁言等信息
  16. /bal(ance)top [page] 显示金钱榜
  17. /mail 查看你的邮件
  18. /suicide 自杀
  19. /afk 暂时离开
  20. /bal(ance) 显示你的余额
  21. /hat 将你手上的东西戴到头上
  22. /getpos 获取你当前的位置
  23. /compass 描述你的方位
  24. /sethome [name] 在你当前位置设置一个家
  25. /home <name> 传送到你设置的家
  26. /delhome <name> 删除一个家
  27. /pay <player> <acount> 向指定玩家打钱
  28. /time 显示当前时间
  29. /exp 显示你当前的经验值
  30. /depth 计算你当前Y坐标相对于海平面的位置
  31. /msg <player> <message> 向指定玩家发送私聊
  32. /me <message> 用第三人称描述一件事
    如:*Im_Explorer_: example
  33. /tpaall 请求所有玩家传送到你这里
  34. /kit 查看你所所拥有的礼包
  35. /helpop 向在线的管理发送信息
  36. /back 传送到你上一次保存的地点(如死亡地点，接受传送的地点)
  37. /skills 打开技能菜单
  38. /enchantinfo 查看Eco更多附魔里的附魔属性
  39. /pets 打开宠物菜单
  40. /jobs 打开职业菜单
  41. /ce 打开CrazyEnchants菜单
    如果以上指令出现未知命令或者无权限的情况，请联系管理或者反馈至xiecraft@gmail.com

Part3 领地

创建领地:
首先，先用木棍左右键选取一个选区，然后在聊天栏里输入
/res create <name>
即可(1 游戏币/Block)
删除领地:
站在要删除的领地内然后输入 /res remove
如果不想到领地内也可以输入 /res remove <name>
最后输入 /res confirm 即可
设置玩家权限:
如果您是领地创建者，那么您不需要更改任何权限
如果您要给您信任的朋友权限，可以使用 /res padd <player> 来给予您的朋友基础权限
或使用 /res pset <player> 来手动设置您的朋友的每一个权限
设置全局权限:
使用 /res set [权限] 来设置领地的全局权限
如果不填写权限名，将会打开领地权限的GUI
更多请看领地插件的自带帮助(直接输入指令不带参数即可)

Part4 强化

如何强化:
将炉岩碳或者高级炉岩碳放在熔炉燃料位，武器放在物品位即可
(请一定使用原版熔炉)

强化成功几率请看服务站门口全息显示
强化保护券:
在+11以后，装备若强化失败，将会消失，强化保护券可以使装备不消失
使用方法:在强化时放于背包即可
